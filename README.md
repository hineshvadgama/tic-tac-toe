# tic-tac-toe

A 2 player tic-tac-toe game that I implemented using HTML, CSS, JavaScript with React.

[View the live app here](https://tictactoe-94.netlify.app/)

# Installation

Once inside the repository folder, install any dependencies

`yarn install`

To run the project:

`yarn start`
